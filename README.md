# Event Management System

A full-stack event management application with a React frontend and Node.js/Express backend.

## Project Structure

- `backend/` - Express server, MongoDB models, authentication routes
- `frontend/` - React app for organisers and participants

## Setup

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Notes

- The backend runs from `backend/server.js`
- The frontend runs via `react-scripts start`
- `.gitignore` already excludes `node_modules`, environment files, and build artifacts
